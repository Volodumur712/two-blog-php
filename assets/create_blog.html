Створення по черзі як в останньому уроці

Перші кроки по створенню сайта

  1. Дизайн є
  2. Спроектована база даних
  3. План реалізації
  • Встанови та налаштуй XAMPP.
  • Відредагувати файли через блокнот
  • Створи в корені папки віртуального сервера файл index.php

      • Налаштування БД

    • Зайди в phpmyadmin та створи базу даних
    • Створи таблицю (posts)
    • При створенні бази даних необхідно вказувати кодування utf8-general_ci.
    • Під час створення таблиці вказувати назви потрібно латиницею, у нижньому регістрі букв.
    • поле ID тип INT і галочку автопродовження (Чек бокс з підписом «A_I»)
    • поле назв і іншого тексту VARCHAR 255
    • структуру бази даних проекту можна намалювати на ресурсі draw.io
    • він же https://app.diagrams.net/
    • можна загуглити як робити нормальні форми баз даних
  • Завантаж файл з темплейтом собі на ПК.
  • Налаштуй віртуальний хост php-blog.test та розверни на ньому тему, яку ти завантажив.
  • Налаштуй структуру проекту та шляхи до файлів по прикладу з відео.
  • структурувати папки
  • Винеси код для посту в окремий файл post.php.
  • Розбий проект на частини(partials).
  • Для підключення до MySQL із PHP нам треба вказати налаштування підключення: адресу сервера, логін, пароль, назву бази даних
  • додати логіку навігації по сторінках і частинах
